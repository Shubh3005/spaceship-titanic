# CMPSC 448 Honors Option Project: Spaceship Titanic

## Overview
Binary classification challenge from [Kaggle Spaceship Titanic](https://www.kaggle.com/competitions/spaceship-titanic).

Predict which passengers were transported to an alternate dimension during a spaceship collision.
